# README
## 初始化
1. 打开termux终端，没有请到[这](https://f-droid.org/repo/com.termux_117.apk)下载
2. 输入以下命令
```console
$ pkg install git
$ pkg install python
$ git clone https://github.com/andogy/caibi-shell/
```
## 使用
### 压测脚本
1. 输入以下命令
```console
$ cd ./caibi-shell
$ bash ./cc.sh
```
2. 接下来输入shell要求的参数即可，对象网址请加上http://
### 单页爬站
1. 输入以下命令
```console
$ cd ./caibi-shell
$ python ./spider.py
```
2. 输入网址，回车，等待结果即可
